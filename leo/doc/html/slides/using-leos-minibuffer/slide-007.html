Several of Leo’s commands use the minibuffer to get arguments.

For example, Leo’s find command accumulates the search and replace strings in the minibuffer.

To invoke the find command, type <Ctrl-F>. This puts the focus in the minibuffer and shows the Find tab. Important: the Find tab just shows you the status of search and replace operations. You control those operations from the minibuffer.
