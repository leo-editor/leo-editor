Indentation, 2Â¶

Indentation is controlled only by the indentation of references. The outline level of nodes in expansions does not affect indentation in any way.

This decoupling of outline structure from indentation is very important: it allows you to create organizer nodes without affecting the external file in any significant way (other than sentinel comments).
